# Third-party assets

All third-party files are bundled locally. The game performs no runtime requests to these sources.

## Textures

- `Rock027` by ambientCG, CC0 1.0: https://ambientcg.com/view?id=Rock027
- `Long White Tiles` by Jenelle van Heerden and Sergej Majboroda, CC0 1.0: https://polyhaven.com/a/long_white_tiles
- `Concrete Floor Worn 001` by Dimitrios Savva and Rico Cilliers, CC0 1.0: https://polyhaven.com/a/concrete_floor_worn_001

## Audio

- `Breathing Tired` by artisticdude, CC0 1.0: https://opengameart.org/content/breathing-tired

## Visual reference

- `Smiler (Backrooms)` by Edward Johnson 3 / lucaarcadu, CC BY 4.0: https://sketchfab.com/3d-models/smiler-backrooms-ea10d7f4750341f0ada6a12a6530014a

  The downloadable archive requires an authenticated Sketchfab session, so the game uses an original optimized local mesh while preserving the credited model as its visual reference.
